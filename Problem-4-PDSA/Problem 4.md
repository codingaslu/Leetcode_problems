Write a function **expanding(L)** that takes a list of integer `L` as input and returns `True` if the absolute difference between each adjacent pair of elements strictly increases.

**Sample Input**

```
[1,3,7,2,9]
```

**Output**

```python
True
```

**Sample Input**

```
[1,3,7,2,-3]
```

**Output**

```python
False
```
